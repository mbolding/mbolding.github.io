Card Game Rules

General Rules

  1. Objective: Form a complete set of cards, one from each category.
  2. Setup: Shuffle the deck and place it face down. Draw cards to form your hand.
  3. Gameplay: Each player can hold up to 4 cards. Players take turns drawing a card from the deck or discarding a card from their hand.
  4. Complete Set: A complete set consists of one card from each category.
  5. Winning: The first player to collect a complete set wins the game.
  6. Restart: If the deck runs out, shuffle the discard pile to form a new deck. Players can restart the game at any time.

Game Categories

EEG

  • Electrodes: Scalp, Intracranial, Dry, Wet
  • Brain Waves: Alpha, Beta, Theta, Delta
  • Processes: Signal Acquisition, Filtering, Amplification
  • Applications: Epilepsy, Sleep Studies, BCI, Cognitive Research

MEG

  • Sensors: SQUID, Optically Pumped, Planar Gradiometer
  • Brain Waves: Alpha, Beta, Theta, Gamma
  • Processes: Magnetic Field Detection, Signal Processing, Source Localization
  • Applications: Epilepsy, Language Mapping, Sensory Processing, Cognitive Research

MRI

  • Imaging Techniques: T1, T2, FLAIR, DWI
  • Applications: Tumor Detection, Stroke Assessment, Brain Structure Analysis
  • Processes: Magnetic Resonance, Image Reconstruction, Contrast Enhancement
  • Equipment: Magnets, Coils, Gradient Systems

PET

  • Tracers: FDG, Oxygen-15, Carbon-11
  • Applications: Cancer Detection, Neurological Disorders, Heart Disease
  • Processes: Positron Emission, Image Processing, Data Analysis
  • Equipment: Scanners, Detectors, Cyclotron
