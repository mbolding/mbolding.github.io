Differential Equations

Chapter 14: Slope Fields & Phase Portraits

An Ordinary Differential Equation (ODE) defines a vector field. Solutions flow along this field. We visualize this with a Phase Portrait.

Phase Space: A plot where axes are state variables (e.g., Position $x$ vs Velocity $v$).
Vector Field: At each point, an arrow points where the state will go next. Instructions: Select a system. Click on the canvas to spawn a trajectory starting from that state.
